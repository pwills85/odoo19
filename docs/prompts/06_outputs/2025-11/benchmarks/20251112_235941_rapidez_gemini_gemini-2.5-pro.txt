En Odoo 19, `t-esc` muestra contenido de forma segura, escapando caracteres HTML para prevenir ataques de Cross-Site Scripting (XSS). Por otro lado, `t-out` inserta el contenido en bruto (raw) sin escaparlo, lo cual es útil para renderizar HTML dinámico que tú controlas. En resumen, siempre debes usar `t-esc` para datos de usuario o no confiables, y `t-out` solo para HTML generado internamente y de confianza.
